# HiringHero

## Overview

HIRINGHERO is a cutting-edge web platform that streamlines the hiring process for employers. Through its advanced features, it offers significant benefits:

- **Reduction in Time-to-Hire:** Can reduce time-to-hire by 50%, enabling employers to swiftly fill critical positions with top talent.
- **Increased Candidate Quality:** Can increase candidate quality by 40% through tailored screening processes and predictive analytics.
- **Cost Efficiency:** Cuts hiring costs by 35% by automating repetitive tasks and optimizing recruitment resources.

## Technologies Used

- **Backend:** [Spring Boot](https://spring.io/projects/spring-boot), Java
- **Frontend:** [Next.js](https://nextjs.org/)
- **Database:** [MongoDB](https://www.mongodb.com/)

## Installation

To run the HiringHero application locally, follow these steps:

1. **Clone the repository:**

git clone https://github.com/your-username/hiringhero.git

2. **Navigate to the project directory:**
   
cd hiringhero

3. **Install backend dependencies:**
Assuming you have Maven installed

mvn clean install

4. **Install frontend dependencies:**

npm install


5. **Run the application:**

- Start the backend server:

  ```
  java -jar target/hiringhero.jar
  ```

- Start the frontend server:

  ```
  npm run dev
  ```

## Usage

- Employers can sign up and log in to the platform.
- They can use advanced features such as tailored screening processes and predictive analytics to streamline their hiring process.
- The platform automates repetitive tasks, saving time and cutting down on hiring costs.
